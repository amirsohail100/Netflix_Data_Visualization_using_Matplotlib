# 🎬 Netflix Content & Maturity Ratings Data Visualization

Is project ka maqsad Netflix ke movies aur TV shows ke data ko analyze aur visualize karna hai. Isme humne khaas taur par content types, regional trends, aur **maturity/explicit ratings** (jaise TV-MA, R, PG-13) par focus kiya hai.
